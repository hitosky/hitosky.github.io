水平居中

1.inline或inline-*元素 
HTML:
<nav>
<a href="#">One</a>
<a href="#">Two</a>
<a href="#">Three</a>
</nav>
			
CSS:
nav{
 text-align:center;
}
nav a {
 text-decoration:none;
}
One Two Three
2.单个块级元素 
HTML:
<main >
 <div>
 I am a block-level centered element.
 </div>
</main>

CSS:
div{
 margin:0 auto;
}
I am a block-level centered element.
3.多个块级元素 
HTML:
<main >
  <div class="child1">
    I am child1 block element;
 </div>
 <div class="child2">
    I am child2 block element;
 </div>
 <div class="child3" >
    I am child3 block element;
 </div>
</main>
		
CSS:
main {
  display:flex;
  justify-content:center;
}
main div {
  width:20%;
  background:#000;
}
I am child1 block element;
I am child2 block element;
I am child3 block element;

垂直居中

1.inline或inline-*元素
1.1 单行 
HTML:
<main>
  <div >
  I am a center line.
  </div>
</main>

CSS:
main div {
  height:60px;
  line-height:60px;
  text-align:center;
}
I am a center line.
1.2多行
方案一:使用flex(同样适用于块级元素) 
HTML:
<div >
 <p>I'm vertically centered multiple lines of text in a flexbox container.</p>
</div>

CSS:
div {
  display:flex;
  flex-direction:column;
  justify-content:center;
}

I'm vertically centered multiple lines of text in a flexbox container.

方案二：使用"ghost"元素 
CSS:
  div {
    position:relative;
  }
  div::before {
    content:"";
    display:inline-block;
    height:100%;
    width:1%;
    vertical-align:middle;
  }
  div p{
  display:inline-block;
  width:200px;
  vertical-align:center;
  }

I'm vertically centered multiple lines of text in a flexbox container.

2.块级元素
2.1已知高度的块级元素 
HTML:
<main >
  <div >
    I am a block-level vertical centered element.
  </div>
</main>
    
CSS:
main{
  position:relative;
  height:150px;
}
main div{
  position:absolute;
  height:100px;
  top:50%;
  margin-top:-50px;
}
I am a block-level vertical centered element.
2.2未知高度的块级元素 
CSS:
main {
  position:relative;
  height:150px;
}
main div{
 position:absolute;
 top:50%;
 transform:translateY(-50%);
}
I am a block-level vertical centered element with unknown height.
