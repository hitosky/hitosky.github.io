ubuntu与windows常见运维

1.主机与虚拟机通信

虚拟机安装ssh，打开22端口，通过NAT模式设置网络，主机通过winscp或者putty访问虚拟机

2.主机与虚拟机共享文件夹

虚拟机（重新）安装vmware tools，然后安装open-vm-dkms，在虚拟机VM->Settings->Options->Shared Folders下设置主机共享文件夹，完事后虚拟机终端输入命令： sudo mount –t vmhgfs .host:/ /mnt/hgfs
